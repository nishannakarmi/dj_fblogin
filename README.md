# dj_fblogin
A simple facebook login app using django-allauth

SetUP:
- Create a virtualenv using python3
- Install python packages using command pip install -r requirements.txt
- Create a facebook app and create a facebook social app from django backend. Insert the app ID and app secret to the social app.
- Run the django development server using python manage.py runserver
- Go to the url http://localhost:8000/
